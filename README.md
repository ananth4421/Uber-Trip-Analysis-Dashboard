# Uber Trip Analysis Dashboard

## Project Overview
This project analyzes Uber trip data using Power BI to provide business insights through interactive dashboards.

## Tools Used
- Power BI
- Microsoft Excel

## Dashboard Features
- Total Trips
- Total Booking Value
- Trip Distance
- Trip Duration
- Vehicle Type Analysis
- Payment Method Analysis
- Pickup and Drop-off Analysis
- Interactive Filters and Slicers

## Skills Demonstrated
- Data Cleaning
- Data Visualization
- Dashboard Design
- KPI Reporting
- Business Analysis
